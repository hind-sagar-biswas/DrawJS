# DrawJS
Vanilla JavaScript Drawing Application

A drawing application made using HTML5 `<canvas>` element and Vanilla JavaScript

## Features

* Draw Anything

* Choose any color from color picker

* Change the line width

* Save drawing as `.PNG` image

* Clear Board
